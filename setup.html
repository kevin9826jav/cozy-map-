🔧 Setup Guide

Get your Cozy Travel Progress app up and running

← Back to Demo View Full App
1 Create Google Cloud Project
  1. Go to Google Cloud Console
  2. Click "Select a project" at the top
  3. Click "New Project"
  4. Enter a project name (e.g., "Travel Progress App")
  5. Click "Create"
2 Enable Required APIs
  1. In your project, go to "APIs & Services" → "Library"
  2. Search for and enable these APIs:
    • Maps JavaScript API
    • Places API
  3. Click "Enable" for each API
3 Create API Key
  1. Go to "APIs & Services" → "Credentials"
  2. Click "Create Credentials" → "API Key"
  3. Copy the generated API key
  4. Important: Click on the API key to configure restrictions
⚠️ Security Note: Always restrict your API key to specific domains or IP addresses to prevent unauthorized usage.
4 Restrict API Key (Recommended)
  1. Click on your API key in the credentials list
  2. Under "Application restrictions", select "HTTP referrers"
  3. Add your domain (e.g., localhost/* for testing)
  4. Under "API restrictions", select "Restrict key"
  5. Select only the APIs you enabled (Maps JavaScript API, Places API)
  6. Click "Save"
5 Update Your App

Replace YOUR_API_KEY in your index.html file with your actual API key:

<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_ACTUAL_API_KEY&libraries=places"></script>

For example:

<script src="https://maps.googleapis.com/maps/api/js?key=AIzaSyB...&libraries=places"></script>

Enter your API key below to test if it's working:

7 Run Your App

Once your API key is working:

  1. Open index.html in your web browser
  2. Or use a local server for better experience:
    # Python 3 python -m http.server 8000 # Node.js npx serve . # PHP php -S localhost:8000
  3. Visit http://localhost:8000 in your browser
🎉 Congratulations! Your Cozy Travel Progress app should now be working with Google Maps!
